**github设置显示语言**

在项目根路径下建立一个文件，命名为`.gitattributes`，添加以下文件

```bash
*.js linguist-language=java
*.css linguist-language=java
*.html linguist-language=java
```

***.js linguist-language=此处填写你自己的编程语言**
